About Movie Explorer

Movie Explorer is a project that allows users to browse popular movies, check ratings, explore genres, and suggest new films to be added. This project demonstrates responsive design, dynamic JavaScript features, and accessibility best practices.

Features

  • Browse a curated list of popular movies
  • View detailed movie information in modals
  • Submit suggestions for new movies via a form
  • Dynamic content loaded from JSON data
  • Weather info included in the footer for fun

About the Developer
